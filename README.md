# Customer Intelligence System

An end-to-end Machine Learning system that combines **Customer Segmentation** and **Churn Prediction** to generate **actionable business insights** through an interactive dashboard.

---

## Live Demo

https://customer-intelligent-system.streamlit.app

---

## Project Overview

This project goes beyond traditional ML models by integrating:

* **Customer Segmentation (KMeans Clustering)**
* **Churn Prediction (Logistic Regression & Random Forest)**
* **Smart Business Insights (USP)**
* **Interactive Dashboard using Streamlit**

---

## Key Features

* Visualizes customer clusters
* Predicts churn probability for each customer
* Compares multiple ML models
* Automatically selects best-performing model
* Generates business insights for decision-making
* Predicts churn for new customers (interactive input)
